# Taniti Static Website Prototype

This is a simple multi-page static website built for a school UX project.
It uses plain HTML and CSS only, with one shared stylesheet and no frameworks.

## Files

- `index.html` — Home page
- `things-to-do.html` — Things to Do page
- `lodging.html` — Lodging page
- `food-dining.html` — Food & Dining page
- `transportation.html` — Transportation page
- `travel-info.html` — Travel Info page
- `faqs.html` — FAQs page
- `styles.css` — shared stylesheet for all pages

## How to use

1. Open `index.html` in a web browser to view the site locally.
2. All pages are linked from the top navigation bar.
3. Edit the HTML files or `styles.css` to update content and styling.

## GitHub Pages

To publish this site with GitHub Pages:

1. Create a new GitHub repository.
2. Add these files to the repository.
3. In the repository settings, enable GitHub Pages and set the source to the `main` branch (or `master`) and the root folder.
4. Visit the published URL shown in GitHub Pages settings.

## Notes

- This project is designed for easy editing and learning.
- No JavaScript is required for the page navigation.
- The responsive layout is desktop-first but works on smaller screens.
